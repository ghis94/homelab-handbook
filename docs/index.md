# homelab-handbook

Bienvenue dans le handbook technique du homelab.

## Objectif

Avoir une source unique, versionnée et exploitable par IA pour :
- l’infrastructure homelab
- Home Assistant / chauffage
- les procédures d’exploitation
- les incidents et décisions techniques

## Démarrage recommandé

1. Lire la page **Knowledge Base / Méthode**
2. Appliquer la structure de dossiers proposée
3. Utiliser le template de fiche pour documenter proprement
4. Versionner chaque changement via Git
