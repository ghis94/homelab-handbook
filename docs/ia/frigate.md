# Frigate

## Contexte
- Utilisé pour détection vidéo locale et alerting.
- Objectif: détection fiable + faible bruit + exploitabilité domotique.

## Configuration
- Stack IA locale (caméras + pipeline de détection).
- Intégration dans l’écosystème homelab pour notifications et automatisations.

## Problèmes connus
- Faux positifs selon angle/cadrage/lumière.
- Charge machine si profils de détection trop agressifs.

## Solutions
- Ajuster zones, seuils et classes détectées.
- Prioriser qualité de placement caméra avant tuning logiciel.
